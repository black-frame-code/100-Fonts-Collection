## 📖 About

**100 Fonts Collection** is a modern showcase featuring **100 popular fonts** for web designers and developers.

It provides categorized font previews, making it easy to compare typography styles before using them in your own projects. The collection includes Sans-Serif, Serif, Monospace, Display, Handwriting, and Persian fonts.
